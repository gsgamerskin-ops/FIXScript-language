##  About

FIXscript is a lightweight experimental scripting language with:

- Printf system
- Function support
- Libraries (.FXL)
- Project files (.FXP)
- Simple interpreter in Python

---

##  File types

- `.FIX` → source code  
- `.FXL` → libraries  
- `.FXP` → project files  

---

##  Example

```fixscript
function Main()
{
    Printf "Hello FIXscript"
}
```
## Note

This is not a copy of other languages. I was not aware that another FIXScript exists at the time. This project is independent and was created as an original experiment based on inspiration from scripting language design.
