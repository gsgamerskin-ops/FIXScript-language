##  About

FIXscript is a lightweight experimental scripting language with:

- Printf system
- Function support
- Libraries (.FXL)
- Project files (.FXP)
- Simple interpreter in Python

---

##  File types

- `.FIX` → source code  
- `.FXL` → libraries  
- `.FXP` → project files  

---

##  Example

```fixscript
function Main()
{
    Printf "Hello FIXscript"
